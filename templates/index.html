Discussion Tracker

Automate your Business discussion forum audits. Instead of manually searching Canvas to find out which learner has raised a query and whether a TA has replied, this tool pulls all the activity instantly.

How it works

  • 1
    Fetch Queries: Connects to Canvas and pulls all posts.
  • 2
    Track Responses: Identifies TA replies and calculates durations.
  • 3
    Analyze SLAs: Checks if the reply was within the time frame.

Generate Report

Enter the course codes below to generate your comprehensive audit report in Excel.

What is a course code?
The course code is the SIS ID from Canvas. You can enter multiple IDs separated by commas. Note: This tool only processes Business (BUS) courses.

Example:
BUS-910-UC5--Spring-2026
